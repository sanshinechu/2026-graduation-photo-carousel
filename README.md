# 2026 畢業生照片輪播

這是一個可部署到 GitHub Pages 的靜態照片輪播網站。

## 功能

- 自動照片輪播
- 上一張 / 下一張
- 依資料夾篩選照片
- 縮圖快速跳轉
- 隨機播放
- 播放速度調整
- 全螢幕展示
- 鍵盤操作：左右方向鍵換照片、空白鍵播放/暫停

## 使用方式

直接開啟 `index.html` 即可觀看。

GitHub Pages 部署後，首頁會自動讀取 `photos.js` 裡列出的 JPG、JPEG、PNG 圖片。

## 注意

目前瀏覽器與 GitHub Pages 不穩定支援 HEIC 圖片，所以 HEIC 原檔會保留在專案中，但不列入輪播。若要顯示 HEIC，請先轉成 JPG，再重新產生 `photos.js`。
